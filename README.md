# Название проекта
### Мобильное приложение по практике (КСК управление снабжениме)
# Описание проекта
### КСК управление снабжениме - мобильное приложение для автоматизации процесса оформления и обработки заявок на оборудование внутри предприятия. Упрощает взаимодействие между заказчиками и менеджерами, обеспечивая прозрачный контроль на всех этапах снабжения.
# Установка
### Склонировать репозиторий через Android Studio или другими средствами. И запустить его через эмуляцию Android
# Использлование (руководство пользователя)
### Когда приложение открылось надо ввести свой логин и пароль и нажать на кнопку «Войти»
![Add_images](https://github.com/Tralbreaker001/mobile_app/blob/project/images/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-02-23%20203335.png?raw=true)
### Откроется главная страница заказщика, от сюда можно перейти на страницу создания заяки на закупку оборудовнаия и посмотреть уже созданные тобой заявки, чтобы проверить какой у них статус выполнения
![Add_images](https://github.com/Tralbreaker001/mobile_app/blob/project/images/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-02-23%20203820.png?raw=true)
### При нажатие на кнопку «Создать заявку» вы переходите на страницу создания заявки на закупку оборудования. Здесь нужно ввести название оборудования, его описание и какое количество вам нужно. После нужно нажать на кнопку «Создать», после чего вас перекинет на главную страницу и снизу будет уведомление, что заяка создана.
![Add_images](https://github.com/Tralbreaker001/mobile_app/blob/project/images/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-02-23%20204731.png?raw=true)
### При нажатие на кнопку «Мои заяки» вы перейдете на страницу со всеми заявками на закупку, которые вы создали. Снизу каждой заяки будет указан текущий статус её выполнения.
